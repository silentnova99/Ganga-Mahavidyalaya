# Ganga Mahavidyalaya

A modern, responsive college website for Ganga Mahavidyalaya.

## Features

- 🏛️ Complete college website with all necessary sections
- 📱 Fully responsive design
- ✨ Dynamic animations and interactions
- 🖼️ Image gallery with lightbox
- 📊 Counter animations
- 📱 Mobile-first approach

## Sections

- Home
- About College
- Courses & Departments
- Admissions
- Faculty
- Activities & Events
- Committees
- Facilities
- NAAC
- Placement
- Contact

## Deployment

This website is ready for GitHub Pages deployment.

## Local Development

```bash
# Serve locally
python -m http.server 8000
# or
npx serve
```

## Technology Stack

- HTML5
- CSS3 (Custom properties, Responsive Grid)
- JavaScript (Vanilla JS)
- Google Fonts
- Material Symbols Icons
